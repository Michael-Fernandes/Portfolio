Interactive LookBook:
A Rapid Prototype of a Novel Interaction

Inspiration

In a discussion with a friend, I generated this idea as a new way to interact with E-commerce websites. A way to present users with full wardrobes (or image compositions with products) and allow them to hover on individual items to find out more information.

When I attempted to communicate the idea in verbal conversation, I felt unsure of how effective the interaction might be. It seemed like a good idea in my head but, without actually having something to play around with, I was unsure how intuitive and effective it would be.

I decided to rapid prototype the idea.

Design Ideation

The process began by creating quick wireframe sketches. I labeled how the page would work together and identified the minimum components the prototype would need to communicate the idea.

To represent how a online shopper might use this type of interface, I included representations of what this specific interaction could look like in the context of a larger E-commerce website.

Proof of Concept

I selected a single "wardrobe" page and a "quick view" component to prototype using the following tools:

  • SVG code generated in Adobe Illustrator
  • D3.js, data visualization library
  • jQuery, javascript library

How I built the prototype

  • Outlined articles of clothing in llustrator
  • Exported outlines into SVG file format
  • Created a HTML/CSS webpage and placed SVG such that it covered the image of the wardrobe.
  • Developed D3.js code to interact with the SVG outlines
  • Progmatically generated the quick view of each clothing item using jQuery

Evaluation

I tested it with a few test users and the result were promising! I found users felt familiar with the prototype and even found users actually told me in post test interviews that an interactive lookbook is something they have wanted in the past.

Design Iteration

With the feedback gathered in the test, I used Adobe XD to rapidly iterate layouts on how an interactive lookbook would appear in an E-commerce website.

I next created a series of wireframes and high fidelity mockups to further articulate my design idea.

Development

In the future, I plan to develop this prototype into working componenets in React.
